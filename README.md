# 🏥 Clinic Management System (Console App)

A simple Clinic Management System built using **C#** to practice **Object-Oriented Programming (OOP)** concepts.

---

## 🎯 Project Goal

This project was created to strengthen my understanding of:

- Classes & Objects
- Inheritance
- Encapsulation
- Basic CRUD operations
- Console-based user interaction
- Input validation using TryParse

---

## ⚙️ Features

✔ Add Doctor  
✔ Show All Doctors  
✔ Add Patient  
✔ Show All Patients  
✔ Book Appointment  
✔ Cancel Appointment  
✔ Show All Appointments  
✔ Show Doctor Appointments  

---

## 🧠 Concepts Used

- OOP Design (Separation of concerns)
- Data validation
- Lists and collections
- Control flow (switch, loops)
- Error handling (user input)

---

## 🖥️ How It Works

The system runs in the console and provides a menu for the user to:

1. Manage doctors and patients  
2. Book and cancel appointments  
3. View data in an organized way  

---

## 📌 Sample Menu

```
1- Add Doctor
2- Show All Doctors
3- Add Patient
4- Show All Patients
5- Book Appointment
6- Cancel Appointment
7- Show All Appointments
8- Show Doctor Appointments
9- Exit
```
